Set up a new Roblox game project from scratch. Steps: create Rojo project (`default.project.json`), create src/ folder structure (ServerScriptService, ReplicatedStorage, StarterPlayerScripts, StarterGui), configure Selene (`.selene.toml`), set up Luau LSP config, create base modules (Config.lua, RemoteSetup.lua, AssetConfig.lua), initialize GitHub repo with CI workflow.

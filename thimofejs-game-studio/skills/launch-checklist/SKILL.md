description: Post-deploy launch checklist — verify game is live and playable, community shout posted, Day-1 KPI baseline captured, monitoring active
---

Run the post-deploy launch checklist: (1) game is public and joinable on Roblox, (2) thumbnail and description display correctly, (3) Gamepasses visible in game store, (4) community-manager Group shout posted, (5) kpi-tracker Day-0 baseline captured, (6) 30-day monitoring window started. All 6 must pass within 2 hours of deploy.

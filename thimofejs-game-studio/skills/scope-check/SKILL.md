---
name: scope-check
description: Detect scope creep against the locked GDD — flag features not in spec, estimate schedule impact, recommend keep/cut/defer
---

Compare current build state against the locked GDD. Flag any feature not in the GDD. For each flagged item: estimate schedule impact, recommend KEEP (in original scope) / CUT (out of scope) / DEFER (post-launch update). Escalate to CEO only if scope change affects KPI projections.

---
name: asset-audit
description: Audit all Roblox assets in a game for free-distribute license compliance, IP violation risk, and audio copyright — must pass before TOS gate
---

Audit every asset in the game: verify free-distribute license via Roblox API, check for IP/trademark violations, verify audio is Roblox Marketplace (no external URLs), confirm no NSFW or violent content. Output: PASS / FAIL per asset, with flagged assets listed. All flags must be resolved before tos-check.

name: system-audit
description: Audit a completed pipeline cycle for process violations, skipped gates, stage timings, and bottlenecks — monthly report with specific improvements
---

Audit a completed game pipeline cycle. Verify every mandatory stage was completed in order with proper sign-offs. Identify any skipped quality gates (especially tos-guard and qa-lead). Measure time per stage vs targets. Identify cycle bottleneck. Deliver audit report to CEO with: violations found, velocity metrics, top 3 process improvement recommendations.

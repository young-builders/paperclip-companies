name: ab-test
description: Design and track A/B experiments — thumbnail CTR, Gamepass prices, FTUE flows, live event offers — with statistical validity checks and documented winners
---

Run a structured A/B test. Define: hypothesis, control (A), variant (B), success metric, minimum sample (≥500 per variant), duration. Track results. Conclude only when minimum sample reached AND ≥5% relative difference observed. Document winner, propagate as new default. Max 2 simultaneous tests per game.

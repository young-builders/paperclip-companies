Define 3-5 game pillars that will guide all creative and technical decisions. Each pillar must be falsifiable, create tension with alternatives, and include a design test showing how it resolves a real decision. Also articulate the core fantasy, unique hook, target MDA aesthetics, and anti-pillars.

## Deliverables
- Game pillars document with design tests
- Core fantasy statement
- MDA aesthetic priority ranking
- Anti-pillars list

Take a game concept from initial idea through the first three development phases — Ideation, Pre-Production, and Prototyping — resulting in a validated playable prototype ready for production.

## Milestones

1. **Concept Lock** — Game pillars defined, core fantasy articulated, competitive positioning mapped
2. **Pre-Production Complete** — Systems decomposed, GDDs drafted for core mechanics, architecture planned
3. **First Playable** — Prototype validates the core mechanic, findings documented, go/no-go decision made
